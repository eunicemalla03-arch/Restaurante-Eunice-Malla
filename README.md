# Sistema de Gestión de Restaurante

## Estudiante

## Descripción
Este proyecto implementa un sistema básico de gestión de restaurante utilizando Programación Orientada a Objetos en Python.

Permite registrar productos y clientes mediante una estructura modular organizada en carpetas.

## Estructura del proyecto

restaurante_app/
│
├── modelos/
├── servicios/
└── main.py

## Tipos de datos utilizados

- str → nombres y correos
- int → edad y cantidades
- float → precios
- bool → estados de disponibilidad
- list → almacenamiento de clientes y productos

## Reflexión

El uso de identificadores descriptivos mejora la comprensión del código y facilita el mantenimiento del proyecto. Los tipos de datos permiten representar correctamente la información y las listas ayudan a administrar múltiples objetos dentro del sistema.
